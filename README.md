AirCanvas is a virtual painting application that lets you draw in real-time using hand gestures, powered by **MediaPipe** and **OpenCV**. You can select colors, undo/redo strokes, save your canvas, and even clear the screen‚Äîall without touching the mouse or keyboard.

---

## Features

- Real-time hand-tracking drawing using **MediaPipe Hands**
- Multiple color options: **Red, Green, Blue, Yellow**
- Undo and redo functionality
- Save your artwork as a PNG file
- Clear canvas with a hand gesture
- Session saving/loading (future feature)
